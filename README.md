# Operating Systems

This repository contains my projects and assignments for the Operating Systems course.

## Contents

- **Assignment 2** — Peterson locks and tournament tree lock
- **Assignment 3** — Shared memory and multi-process logging

## Projects

Implementation of kernel-level modifications in xv6.

### Assignment 2

Extended xv6 with shared memory syscalls, shared-page tracking, and a multi-process logger.

All projects are based on xv6-riscv and were developed as part of the Operating Systems course.
